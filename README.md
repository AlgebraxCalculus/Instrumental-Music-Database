# Instrumental-Music-Database
Project for Multimedia Database
---
Design and implementation of a database system for music retrieval based on audio content.
1. Construct or collect an audio dataset consisting of at least 500 short audio files of instrumental music. The audio files must have a suitable duration for feature extraction, and each file should represent a musical piece with a distinct melody. (Students may freely choose the audio file format.)
2. Design a set of features to identify and characterize the musical pieces in the collected audio dataset.
The feature set must include attributes that: capture the similarities among musical excerpts, and represent the differences among musical excerpts.
Clearly explain the rationale for selecting these features and analyze the information value provided by each selected feature.
3. Develop a database system to manage the available metadata and describe the mechanism for retrieving similar musical pieces based on this metadata.
4. Implement a music retrieval system in which: the input is an audio file containing a musical excerpt that may be present or absent in the dataset, and the output is the five most similar audio files, ranked in descending order of content similarity with respect to the query audio.
a. Present the system block diagram and describe the processing workflow used to fulfill the requirements of the assignment.
b. Present the intermediate results produced during the music retrieval process.
5. Demonstrate the system and evaluate the achieved results.
